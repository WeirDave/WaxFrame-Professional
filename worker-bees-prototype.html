Design notes (will be deleted with this file once the real page ships):
  • One row open at a time by default — click another row, the previous one collapses. Keeps the page tidy on laptop screens.
  • Collapsed row is roughly the size of today's Worker Bees row. No extra vertical density baseline.
  • Tier cards marked is-active when the AI's current model matches that tier's pick. Click any other card to swap (demo only here).
  • NONE-tier slots (e.g. DeepSeek's "no thinker") render disabled instead of hidden — users see the gap.
  • "Why each pick" is a second click inside the expanded row. Optional context for users who want to learn the classifier's reasoning. Stays hidden by default.
  • Hive Profile dropdown at the top orchestrates across all rows. When a profile is active and a row's tier-matched model is the current pick, the row shows a small profile tag (e.g. 🧠 Heavy thinkers pick). Manual override on a single row is allowed and tagged.
