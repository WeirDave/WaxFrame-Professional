What is an API key and why do I need one?

In Pro mode, AI Hive sends your prompts directly to each AI and fills in the responses automatically — no copy/paste, no tab switching. An API key is a private password that lets AI Hive talk to each AI on your behalf.

Without a key, AI Hive works in Free mode — you copy/paste prompts manually. API keys are only needed for the fully automated Pro experience.

🔒 Your keys never leave your device

Keys are stored in your browser's localStorage only. They go directly from your browser to each AI provider. AI Hive has no server — your keys are never shared with any third party.

How to get your key — step by step
ChatGPT (OpenAI)
Pay-as-you-go
Model:gpt-4oOpenAI's flagship model
  1. Go to platform.openai.com/api-keys and sign in.
  2. Click Create new secret key, give it a name, click Create.
  3. Copy the key immediately — shown only once.
  4. Paste it into the ChatGPT field in AI Hive and press Enter.
⚠️ Requires credit balance. Go to Billing → Payment methods to add credit before API calls will work.
Claude (Anthropic)
Pay-as-you-go
Model:claude-opus-4-5Anthropic's capable everyday model
  1. Go to console.anthropic.com/settings/keys and sign in.
  2. Click Create Key, give it a name (e.g. "AI Hive"), click Create.
  3. Copy the key immediately — shown only once.
  4. Paste it into the Claude field in AI Hive and press Enter.
⚠️ Requires credit balance. Go to Settings → Billing and add credit (~$5 minimum). You're only charged for what you use.
Copilot (Microsoft)
Not yet available for individuals
Status:API in Private Preview — enterprise only as of early 2026
ℹ️ Copilot API is not currently available for personal Microsoft 365 subscribers. It requires an enterprise add-on license (E3/E5 + Copilot add-on) and is still in Private Preview. This is a Microsoft restriction — not an AI Hive limitation. We'll add full support the moment Microsoft opens the API to individuals.
✅ Copilot works great in free/manual mode. Open copilot.microsoft.com in a tab, paste your prompts, and copy responses back into AI Hive. Copilot thinks differently from ChatGPT — it's worth keeping in your hive.
DeepSeek
Very affordable
Model:deepseek-chatDeepSeek's flagship model
  1. Go to platform.deepseek.com/api_keys and sign up.
  2. Click Create new API key and copy it.
  3. Paste it into the DeepSeek field in AI Hive and press Enter.
✅ DeepSeek is often 10-20x cheaper per token than OpenAI or Anthropic. Requires a small credit balance (~$5) to get started. Excellent Builder choice if you want to keep API costs low.
Gemini (Google)
Free tier available
Model:gemini-1.5-flashGoogle's fast multimodal model
  1. Go to aistudio.google.com/apikey and sign in with your Google account.
  2. Click Create API Key and copy the key shown.
  3. Paste it into the Gemini field in AI Hive and press Enter.
✅ Gemini is completely free — no credit card or billing required. A set number of requests per day at no cost. Most AI Hive sessions will stay comfortably within the free limit.
Grok (xAI)
Limited availability
Model:grok-beta
  1. Go to console.x.ai and sign in with your X account.
  2. Navigate to API Keys and generate a new key.
  3. Paste it into the Grok field in AI Hive and press Enter.
⚠️ Requires credit balance — add via the billing section at console.x.ai.
Perplexity
Pay-as-you-go
Model:llama-3.1-sonar-large-128k-online
  1. Go to perplexity.ai/settings/api and sign in.
  2. Click Generate under API Keys and copy the key.
  3. Paste it into the Perplexity field in AI Hive and press Enter.
⚠️ Requires credit balance — add via the billing section on the same settings page.
