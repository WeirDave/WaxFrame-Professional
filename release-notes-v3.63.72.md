# Builder Roster: Three Columns for Normal Hives

This tightens the v3.63.71 Builder roster scaling so the common 7-9 AI setup looks visibly richer.

---

## What Changed

**7-9 Builders now use three columns**

Normal hives now render as three roomier columns instead of fitting four compact columns across the roster.

**Icons are larger**

Normal Builder tiles now use 42px provider icons with taller tile spacing, making the roster feel more visual and less sparse.

**Laptop behavior was corrected**

The 1366×768 / laptop media rule now preserves the three-column normal roster instead of overriding it back to a tighter auto-fit grid.
