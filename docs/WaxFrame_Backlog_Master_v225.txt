Observations

- 2026-06-04 — The keepExpanded plumbing now extends to BOTH bulk APIs (applyHiveProfile v3.63.155; recommendModelsForAll v3.63.156). Should be standard practice: any bulk operation that calls renderAIRow indirectly needs to pass keepExpanded:false to preserve user-controlled collapse state. Same goes for the silent:true on saveModelForAI when in bulk. Worth a release-check rule: any bulk loop calling saveModelForAI or renderAIRow without opts is suspect.

- 2026-06-04 — The Jamba/Together AI dropdown disappearance was a latent bug from v3.63.153 when the model dropdown moved from the expanded panel to the collapsed row. The expanded-panel buildModelSelector also returns '' on empty list, but in the expanded position users could expand the row and click Recommend Models to populate. With the dropdown on the collapsed row, an empty list = a hole in the row layout. The stale-state fallback closes that gap.
