Template Library

Templates fill in your Project fields with proven starting content for common document types. Pick one, hit Apply, and you land in WaxFrame with the goal, audience, scope, tone, and reference scaffold already populated. Edit anything that doesn't fit your specifics; the hive takes it from there.

Most templates support two paths — Scratch (you have no draft yet; the hive writes one from the scaffold) and Refine (you have a draft; the hive polishes and tightens it without rewriting wholesale). A few are single-path by design — the Quick Start onboarding template is Scratch-only; the platform-specific Review templates (TripAdvisor, Google Maps, Yelp) are Refine-only.

📐 Template data model — for power users editing exported .json

Every template is a JSON object with these top-level fields:

idUnique slug. Used as DOM ids and storage keys. Custom templates auto-generate one.
nameShort display name on the card.
iconSingle emoji shown on the card and pre-filled into the Project field on apply.
categoryOne of: Quick Start, Career & Hiring, Business & Sales, Content & Marketing, Personal & Everyday, Reviews & Recommendations.
descriptionOne-line explanation rendered on the card. Used as fallback when a path has no override.
pathsArray of supported paths. Values: "scratch", "refine", or both.
pathContentObject keyed by path slug. Each value carries the per-path Project fields (see below).

Inside pathContent[path]:

description(Optional) Overrides the top-level description for this specific path.
goalDocTypeWhat kind of document this is (e.g. "Cover letter").
goalAudienceWho the document is for.
goalOutcomeWhat the document should accomplish.
goalScopeWhat's in scope and what's not.
goalToneVoice and tone guidance.
goalNotesHard constraints — things the hive must not do.
refMaterialReference material scaffold — injected as a Reference card on apply (scratch path).
hint(Optional) Array of post-apply hints rendered in the Template Hint banner. Each entry is { field: "...", text: "..." }.
lengthMode(Optional) "hardcap", "range", or absent.
lengthLimit / lengthMin / lengthUnit(Optional) Length-mode parameters.

Custom templates additionally capture hive (the AI lineup + Builder selection at save time) and startingDoc (the source document, when the user opts in). Both ride alongside pathContent.
